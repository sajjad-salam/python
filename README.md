- استخراج ملف الركوايرمنت للمشروع `pipreqs`:
  ```bash
  pip install pipreqs
  ```

##  تحويل ملف بايثون الى ملف مشروع exe للحاسبة
-  تثبت هاي  

``` python
python -m pip install auto-py-to-exe
```


-  وراهة تشغل  
``` python
python -m auto_py_to_exe
```
